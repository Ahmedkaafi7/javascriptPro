# HornFix - Mobile Repair Website

A responsive mobile repair business website built with HTML, CSS, and JavaScript.

After 3 months of learning JavaScript, this project demonstrates my ability to build interactive and dynamic web applications using my previous knowledge of HTML and CSS.

---

## 🚀 Features

- Responsive Navigation Bar
- Dark Mode (with localStorage)
- Animated Counters
- Dynamic Pricing System (Android / iPhone)
- Discount Plans
- Testimonial Slider
- Contact Form Validation
- Scroll Animations (Intersection Observer)
- Loader Screen
- Back to Top Button

---

## 🛠️ Technologies

- HTML5  
- CSS3 (Flexbox & Grid)  
- Vanilla JavaScript (ES6)  
- Font Awesome
